# Dashboard de Ventas – Super Auto S.A.

## Objetivo del proyecto
Desarrollar un dashboard interactivo en Power BI para analizar el desempeño comercial de la empresa Super Auto S.A., permitiendo visualizar indicadores clave de ventas y apoyar la toma de decisiones del área comercial.

## Contexto de negocio
La empresa cuenta con ventas distribuidas por zonas, vendedores y periodos de tiempo. El objetivo es identificar:
- Zonas con mayor y menor rendimiento
- Ranking de vendedores
- Evolución de ventas en el tiempo

## KPIs y métricas principales
- Total de ventas
- Ventas por zona
- Ventas por vendedor
- Ranking de vendedores
- Tendencia de ventas por periodo

## Herramientas utilizadas
- Power BI
- Microsoft Excel (fuente de datos)
- Modelado de datos

## Resultados e insights
- Se identificaron las zonas con mayor volumen de ventas, lo que permite priorizar estrategias comerciales.
- El ranking de vendedores ayuda a reconocer el desempeño individual y detectar oportunidades de mejora.
- El análisis temporal permite evaluar el crecimiento o caída de ventas por periodo.

## Aprendizajes
- Diseño de dashboards orientados a negocio
- Modelado de datos para análisis comercial
- Uso de Power BI para visualización y análisis de KPIs
